## Task 1: `discPhysics.js` — shared geometry-parameterized solver

**Files:** Create `server/src/games/discPhysics.js`, `server/test/discPhysics.test.js`.

**Interface (Produces):**
```
simulateShot(discs, table) -> { frames, finalDiscs, pocketed, firstContact }
```
- `discs`: `[{ id, ...meta, x, y, vx, vy, r, mass }]` (not mutated). The cue ball has `id: 0`.
- `table`: `{ bounds:{loX,hiX,loY,hiY}, pockets:[{x,y,r}], friction, stopV, restitution, wallRest, maxSteps, frameEvery }`.
- `frames`: arrays of `{ id, x, y }` (rounded), last = resting state. (Color/meta is looked up by id on the client, so frames stay small.)
- `finalDiscs`: survivors at rest.
- `pocketed`: `[{ id }]` removed (in capture order).
- `firstContact`: id of the first non-cue ball the **cue ball (id 0)** touches, else `null`.

**Algorithm** (port Carrom's solver, parameterized; add per-ball-radius walls and firstContact):
```js
export function simulateShot(discs, table) {
  const { bounds, pockets, friction, stopV, restitution, wallRest, maxSteps, frameEvery } = table;
  const live = discs.map((d) => ({ ...d }));
  const frames = [], pocketed = [];
  let firstContact = null;
  const snap = () => live.map((d) => ({ id: d.id, x: Math.round(d.x), y: Math.round(d.y) }));
  for (let step = 0; step < maxSteps; step++) {
    let moving = false;
    for (const d of live) {
      d.x += d.vx; d.y += d.vy; d.vx *= friction; d.vy *= friction;
      if (Math.hypot(d.vx, d.vy) < stopV) { d.vx = 0; d.vy = 0; } else moving = true;
    }
    for (let i = 0; i < live.length; i++)
      for (let j = i + 1; j < live.length; j++) {
        const hit = resolve(live[i], live[j], restitution);
        if (hit && firstContact === null) {
          if (live[i].id === 0 && live[j].id !== 0) firstContact = live[j].id;
          else if (live[j].id === 0 && live[i].id !== 0) firstContact = live[i].id;
        }
      }
    for (let k = live.length - 1; k >= 0; k--)
      if (inPocket(live[k], pockets)) { pocketed.push({ id: live[k].id }); live.splice(k, 1); }
    for (const d of live) bounceWalls(d, bounds, wallRest);
    if (step % frameEvery === 0) frames.push(snap());
    if (!moving) break;
  }
  for (const d of live) { d.vx = 0; d.vy = 0; }
  frames.push(snap());
  return { frames, finalDiscs: live, pocketed, firstContact };
}
```
Helpers `resolve(a,b,e)` (returns true if they were overlapping/collided — same elastic+separation math as Carrom, equal or per-disc mass), `inPocket(d,pockets)` (`hypot(d.x-p.x,d.y-p.y) < p.r` for any pocket), `bounceWalls(d,bounds,wallRest)` (clamp to `bounds` adjusted by `d.r`, invert velocity * wallRest).

**Tests** (`discPhysics.test.js`) — define a small test table inline:
- head-on transfer (struck ball moves forward, ends ahead of striker);
- pocket capture (ball aimed into a pocket is removed);
- friction to rest within cap;
- determinism (two identical runs equal);
- `firstContact` = the id of the first object ball the cue (id 0) hits.

## Task 2: Refactor `carromPhysics.js` to delegate

**Files:** Modify `server/src/games/carromPhysics.js`. (No test file change.)

Keep `BOARD`, `POCKETS`, and `simulateShot(discs)` exported and behaviorally identical. Internally build a `table` from Carrom's constants and call `discPhysics.simulateShot`:
```js
import { simulateShot as solve } from './discPhysics.js';
const TABLE = {
  bounds: { loX: BOARD.inset, hiX: BOARD.W - BOARD.inset, loY: BOARD.inset, hiY: BOARD.H - BOARD.inset },
  pockets: POCKETS.map((p) => ({ ...p, r: BOARD.pocketR })),
  friction: 0.985, stopV: 0.06, restitution: 0.92, wallRest: 0.7, maxSteps: 3000, frameEvery: 2,
};
export function simulateShot(discs) {
  const { frames, finalDiscs, pocketed } = solve(discs, TABLE);
  // Carrom frames historically carry color; re-attach by id so Carrom's client/tests are unaffected.
  const byId = new Map(discs.map((d) => [d.id, d.color]));
  const withColor = frames.map((f) => f.map((d) => ({ ...d, color: byId.get(d.id) })));
  const finals = finalDiscs.map((d) => ({ ...d }));
  return { frames: withColor, finalDiscs: finals, pocketed: pocketed.map((p) => ({ id: p.id, color: byId.get(p.id) })) };
}
```
Note: Carrom's `pocketed` entries were `{id,color}`; preserve that shape. Carrom's `finalDiscs` carried `color`, `r`, `mass` — `discPhysics` preserves all input fields via `{...d}`, so color/r/mass ride through; the `byId` color reattach above is only needed for `frames` (which discPhysics trims to `{id,x,y}`).

## Task 6: `pool.js` — fouls + ball-in-hand

**Files:** Modify `pool.js`, `pool.test.js`.

In `resolveEightball`, compute fouls:
- **scratch**: cue ball potted.
- **wrong first contact**: after assignment, `firstContact`'s group ≠ shooter's group — unless the shooter's group is cleared, in which case the legal first contact is the 8. A no-contact miss (`firstContact === null`) is also a foul (didn't hit anything).
- On any foul: `foul` set, `ballInHand = true` for the opponent, turn passes, the cue ball is restored to the table (re-spot in the kitchen if it was potted). `continues = false`.
- On a clean shot, `ballInHand = false`.

**Tests:** scratch → `foul`, opponent `ballInHand`, turn passes, cue back on table; hitting the opponent's group first (after assignment) → foul + ball-in-hand; a subsequent move with `cue:{x,y}` while `ballInHand` repositions the cue and clears `ballInHand`.
